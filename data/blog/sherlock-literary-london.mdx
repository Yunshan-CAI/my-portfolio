## Project Overview

What if you could step into the shoes of Sherlock Holmes and explore the very streets where his adventures unfolded? Our **interactive literary map** transforms the reading experience by visualizing the geographic elements of Arthur Conan Doyle's detective stories across London.

![Sherlock's Literary London Homepage](/static/images/sherlock-homepage-screenshot.png)

## Implementation

### **Core Features**

- **Interactive Map Interface**: Built with Leaflet.js featuring intuitive navigation and custom zoom controls
- **Multi-layered Information**: Crime scene markers with surrounding witness and police markers using custom SVG icons
- **Story Pathways**: Color-coded markers showing Sherlock's routes through different storylines with dynamic polyline connections
- **Responsive Design**: Optimized for desktop exploration with clear typography and earth-tone aesthetics

### **Technology Stack**

- **Frontend**: HTML5, CSS3, ES6+ JavaScript with module system
- **Mapping Library**: Leaflet.js for interactive map functionality
- **Map Tiles**: ThunderForest Pioneer API for vintage aesthetic

### **User Experience Design**

- **Simplicity First**: Streamlined interface focusing on ease of use with animated homepage transitions
- **Multiple Interaction Methods**: Mouse scrolling, dedicated buttons, and integrated search functionality
- **Bounded Exploration**: Restricted map area keeps users focused on London using Leaflet bounds
- **Progressive Disclosure**: Additional details revealed through marker interactions with custom popup layouts
